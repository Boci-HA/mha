# Manus Home Assistant Addons

A collection of Home Assistant addons powered by Manus AI.

## Addons

### Manus AI Assistant

AI-powered Home Assistant control using Manus - Enable natural language control of your smart home.

**Features:**
- Natural language device control
- Voice command support
- Image recognition and analysis
- AI-powered automation suggestions
- Multi-turn conversations

**Repository:** [manus-ai-assistant](./manus-ai-assistant)

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on store**
2. Click the three-dot menu (⋮) in the top right
3. Select **Repositories**
4. Add this URL: `https://github.com/Boci-HA/mha`
5. Click **Add**
6. Search for "Manus AI Assistant"
7. Click **Install**
